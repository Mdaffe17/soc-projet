
# SOC Project – Wazuh + Suricata + TheHive + Cortex

Ce dépôt contient la construction progressive d'un environnement SOC
(Security Operations Center) basé sur :

- Wazuh (SIEM/XDR)
- Suricata (IPS/NIDS)
- TheHive (SOAR)
- Cortex (Analyse automatisée)

L'objectif est d'apprendre, de documenter et d'automatiser le
déploiement de cette architecture via DOCKER.
